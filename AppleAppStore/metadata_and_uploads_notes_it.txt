NeoOffice è una suite per ufficio completa per OS X. Con NeoOffice, gli utenti possono visualizzare, modificare e salvare documenti OpenOffice, file PDF e la maggior parte dei documenti di Microsoft Word, Excel e PowerPoint.

Gli utenti di NeoOffice 3.x e OpenOffice: prima di acquistare NeoOffice, si consiglia di cliccare sul link «Supporto per NeoOffice» e leggere l'elenco delle modifiche.

NeoOffice è estremamente stabile e migliaia di utenti lo usano giornalmente. Gli ingegneri di NeoOffice hanno creato NeoOffice nel 2003, quando hanno fatto OpenOffice funzionare nativamente su OS X per la prima volta.

Da allora, gli ingegneri di NeoOffice hanno continuamente aggiunti miglioramenti a NeoOffice che gli utenti non troveranno in OpenOffice come ad esempio:

• Aprire Calc o Impress, invece di Writer al inizio con il menu di NeoOffice > Apri al inizio
• Gatekeeper di OS X
• Versioni di OS X
• Evidenziazione nativo OS X
• Supporto per blocco nativo dei file per i volumi locali o di rete
• Sopporto per i Servizi di OS X
• Finestre degli strumenti native flottante


What's New in this Version:
Prima di acquistare NeoOffice, clicca sul link «Sito web di Planamesa Inc.» e prova gratuitamente NeoOffice. This version of NeoOffice adds support for iCloud Drive. We have made signficant changes to NeoOffice's native file locking code to safely coordinate opening, saving, and deleting files in iCloud Drive or files on a shared volume.
