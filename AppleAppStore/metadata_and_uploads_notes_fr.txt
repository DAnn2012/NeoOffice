NeoOffice est une suite bureautique pour Mac qui est basé sur OpenOffice et LibreOffice. Avec NeoOffice, vous pouvez visualiser, modifier et enregistrer des documents OpenOffice, des documents LibreOffice et des documents Microsoft Word, Excel et PowerPoint simples.

Deux ingénieurs ont créé NeoOffice en 2003 quand ils ont fait OpenOffice fonctionner en mode natif sur OS X pour la première fois. Depuis lors, les ingénieurs de NeoOffice ont continuellement ajouté des améliorations à NeoOffice que vous ne trouverez pas dans OpenOffice ou LibreOffice, comme :

• Mode foncé natif
• Menu de NeoOffice > Document par défault pour ouvrir de Calc ou Impress au lieu de Writer au démarrage
• Menu de Fichier > Parcourir toutes les versions pour restaurer les versions précédentes de vos documents
• Verrouillage natif des fichiers pour modifier sans encombre des fichiers dans iCloud Drive, Dropbox ou lecteurs réseau
• Vérification de la grammaire natif Mac
• Surlignage natif Mac
• Prise en charge des Services Mac
• Fenêtres d'outils flottantes natives


What's New in this Version:
Avant d'acheter NeoOffice, s'il vous plaît cliquer sur le lien « Assistance » et lire les notes d'installation.

This version of NeoOffice fixes the following bugs:
• NeoOffice would clear the menubar after selecting the Window > Close menu item
• NeoOffice would crash when printing gradients in certain charts
• The border animation around the most recently copied Calc cells would not automatically stop when copying something in a different document or a different application
• NeoOffice would not display the native macOS tile and tab menu items in the Windows menu
• NeoOffice would sometimes save a document to the file's previous location if the document was moved or renamed in the macOS Finder while open in NeoOffice
• NeoOffice would momentarily freeze while opening or closing a document when running the Mac App Store version of NeoOffice on macOS Big Sur
• NeoOffice would crash due to the LibreOffice bugs #70062, #90606, #100635, #101178, #107786, #108305, #111522,, #114710 #125824, and #128073
