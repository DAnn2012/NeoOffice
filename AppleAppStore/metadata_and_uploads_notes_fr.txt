NeoOffice est une suite bureautique pour Mac qui est basé sur la suite bureautique OpenOffice. Avec NeoOffice, vous pouvez visualiser, modifier et enregistrer des documents OpenOffice et des documents LibreOffice.

Deux ingénieurs ont créé NeoOffice en 2003 quand ils ont fait OpenOffice fonctionner en mode natif sur OS X pour la première fois. Depuis lors, les ingénieurs de NeoOffice ont continuellement ajouté des améliorations à NeoOffice que vous ne trouverez pas dans OpenOffice ou LibreOffice, comme :

• Menu de NeoOffice > Document par défault pour ouvrir de Calc ou Impress au lieu de Writer au démarrage
• Menu de Fichier > Parcourir toutes les versions pour restaurer les versions précédentes de vos documents
• Verrouillage natif des fichiers pour modifier sans encombre des fichiers dans iCloud Drive, Dropbox ou lecteurs réseau
• Vérification de la grammaire natif Mac
• Surlignage natif Mac
• Prise en charge des Services Mac
• Fenêtres d'outils flottantes natives


What's New in this Version:
Avant d'acheter NeoOffice, s'il vous plaît cliquer sur le lien « Assistance : NeoOffice » et lire les notes d'installation.

This version of NeoOffice fixes the following bugs:
• When selecting a menu item while running autospellcheck, NeoOffice would sometimes freeze until autospellcheck finished spellchecking the entire document
• NeoOffice would crash or hang when opening some .rtf files that contain data that does not conform to Microsoft's Rich Text Format specification
• NeoOffice would take 20 - 30 minutes to open spreadsheet documents that contain thousands of unique cell styles
• NeoOffice would sometimes freeze when unlocking a Dropbox or iCloud document in the document's titlebar popup window
• When running NeoOffice on macOS Sierra, both horizontal and vertical scrolling using the trackpad was several times more sensitive than on earlier versions of OS X
• When running NeoOffice on macOS Sierra, NeoOffice would crash when creating a database using the Database Wizard
• NeoOffice would crash if you pressed the Command-w keys while a print dialog was displayed
