NeoOffice est une suite bureautique complète pour OS X. Avec NeoOffice, les utilisateurs peuvent visualiser, modifier et enregistrer des documents OpenOffice, des fichiers PDF et la plupart des documents Microsoft Word, Excel et PowerPoint.

Les utilisateurs de NeoOffice 3.x et OpenOffice : avant d'acheter NeoOffice, nous vous recommandons de cliquer sur le lien « Assistance : NeoOffice » et de lire la liste des modifications.

NeoOffice est extrêmement stable et des milliers d'utilisateurs utilisent quotidiennement. Les ingénieurs de NeoOffice ont crée NeoOffice en 2003 quand ils ont fait OpenOffice s'exécutent en mode natif sur OS X pour la première fois.

Depuis lors, les ingénieurs de NeoOffice ont continuellement ajouté des améliorations à NeoOffice que les utilisateurs ne trouveront pas dans OpenOffice, comme :

• Ouverture de Calc ou Impress au lieu de Writer au démarrage utilisant le menu de NeoOffice > Document par défault
• Versions de OS X
• Vérification de la grammaire natif OS X
• Surlignage natif OS X
• Prise en charge du verrouillage natif des fichiers pour les volumes locaux ou en réseau
• Prise en charge des Services OS X
• Fenêtres d'outils flottantes natives


What's New in this Version:
Avant d'acheter NeoOffice, cliquer sur le lien « Site web : Planamesa Inc. » et essayer NeoOffice gratuitement. This version of NeoOffice adds support for OS X Yosemite's new iCloud Drive feature. We have made signficant changes to NeoOffice's native file locking code to safely coordinate opening, saving, and deleting files in iCloud Drive or files on a shared volume.

This version of NeoOffice also includes fixes for the following bugs:
• NeoOffice would randomly crash when checking if a document has been moved or renamed
• NeoOffice would only display the first image in the many .docx, .xlsx, and .pptx documents
