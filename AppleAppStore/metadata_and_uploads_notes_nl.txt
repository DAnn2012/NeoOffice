Kan Microsoft Office-documenten niet openen? Probeer NeoOffice. NeoOffice kan de meeste Microsoft Word-, Excel- en PowerPoint-documenten te openen, en ook OpenOffice-documenten te openen. Met NeoOffice kunnen gebruikers tekstdocumenten, werkbladen en presentaties lezen, maken en bewerken.

NeoOffice 3.x en OpenOffice gebruikers: wij raden u klikt op de NeoOffice Ondersteuning link en lees de lijst met wijzigingen voordat u een upgrade naar deze versie van NeoOffice.

NeoOffice is zeer stabiel en wordt dagelijks gebruikt door duizenden mensen. De ontwikkelaars van NeoOffice hebben NeoOffice gecreëerd in 2003, toen ze OpenOffice voor het eerst draaiend kregen op OS X.

Sindsdien hebben de NeoOffice ingenieurs - Patrick Luby en Ed Peterlin - voortdurend verbeteringen toegevoegd aan NeoOffice die gebruikers niet in vinden in OpenOffice, zoals:

• Bij starten Calc of Impress tonen in plaats van Writer
• OS X Gatekeeper
• OS X Versies
• OS X-eigen tekstmarkering
• Systeem-eigen ondersteuning voor blokkeren van bestanden op lokale- en netwerkschijven
• Ondersteuning voor OS X Voorzieningen
• OS X-eigen zwevende hulpvensters


What's New in this Version:
This version fixes a bug that would sometimes cause saving a document to be very slow. This bug would occur when the user tried to quit NeoOffice and NeoOffice would ask the user to save any unsaved changes. Saving at that point would be several times slower than normal.
