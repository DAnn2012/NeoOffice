NeoOffice is een compleet pakket kantoorsoftware voor OS X. Met NeoOffice kunnen gebruikers OpenOffice-documenten, LibreOffice-documenten en eenvoudige Microsoft Word-, Excel- en PowerPoint-documenten bekijken, bewerken en opslaan.

NeoOffice is zeer stabiel en wordt dagelijks gebruikt door duizenden mensen. De ontwikkelaars van NeoOffice hebben NeoOffice gecreëerd in 2003, toen ze OpenOffice voor het eerst draaiend kregen op OS X.

Sindsdien hebben de NeoOffice ingenieurs voortdurend verbeteringen toegevoegd aan NeoOffice die gebruikers niet in vinden in OpenOffice of LibreOffice, zoals:

• NeoOffice > Toon bij starten menu voor de opening van Calc of Impress in plaats van Writer bij opstarten
• Bestand > Blader door alle versies menu voor de restauratie van vorige versies van uw documenten
• OS X-eigen bestandsblokkering voor het veilig bewerken van bestanden in iCloud Drive, Dropbox of netwerkschijven
• OS X-eigen grammaticacontrole
• OS X-eigen tekstmarkering
• Ondersteuning voor OS X Voorzieningen
• OS X-eigen zwevende hulpvensters 


What's New in this Version:
NeoOffice, OpenOffice en LibreOffice gebruikers: voordat u koopt NeoOffice, raden we u klikt op de "NeoOffice Ondersteuning" link en lees de lijst met wijzigingen.

This version of NeoOffice fixes the following bugs:
• After starting a slide show from a presentation document that is in full screen mode, exiting the slide show would disable the OS X menubar until the presentation document was closed
• When opening a document while in full screen mode that contains linked but not embedded images or OLE objects, NeoOffice would shrink the newly opened full screen mode window
• When saving certain documents that contain embedded images or OLE objects, NeoOffice would would fail to save one or more of the embedded images or OLE objects
• When running on OS X El Capitan Public Beta, NeoOffice would draw native table headings with the wrong height

This version also fixes the follow security vulnerability. Special thanks go to the security researchers and engineers at LibreOffice.org who identified and provided a fix for this security vulnerability:
• NeoOffice would crash when opening certain Microsoft Word .doc documents
