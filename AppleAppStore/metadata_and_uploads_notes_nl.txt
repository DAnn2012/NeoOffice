NeoOffice is een compleet pakket kantoorsoftware voor OS X. Met NeoOffice kunnen gebruikers OpenOffice-documenten, PDF-bestanden en ook de meeste Microsoft Word-, Excel- en PowerPoint-documenten bekijken, bewerken en opslaan.

NeoOffice 3.x, OpenOffice en LibreOffice gebruikers: voordat u koopt NeoOffice, raden we u klikt op de "NeoOffice Ondersteuning" link en lees de lijst met wijzigingen.

NeoOffice is zeer stabiel en wordt dagelijks gebruikt door duizenden mensen. De ontwikkelaars van NeoOffice hebben NeoOffice gecreëerd in 2003, toen ze OpenOffice voor het eerst draaiend kregen op OS X.

Sindsdien hebben de NeoOffice ingenieurs voortdurend verbeteringen toegevoegd aan NeoOffice die gebruikers niet in vinden in OpenOffice, zoals:

• NeoOffice > Toon bij starten menu voor de opening van Calc of Impress in plaats van Writer bij opstarten
• Bestand > Blader door alle versies menu voor de restauratie van vorige versies van uw documenten
• OS X-eigen bestandsblokkering voor het veilig bewerken van bestanden in iCloud Drive, Dropbox of netwerkschijven
• OS X-eigen grammaticacontrole
• OS X-eigen tekstmarkering
• Ondersteuning voor OS X Voorzieningen
• OS X-eigen zwevende hulpvensters 


What's New in this Version:
Voordat u koopt NeoOffice, klikt op de "Planamesa Inc. website" link en probeert NeoOffice gratis. This version of NeoOffice includes fixes for the following bugs that occur when running on OS X Yosemite:

• NeoOffice would occasionally hang while editing a table in a Base database
• Clicking any hyperlinks that point to a file would fail to open the linked file
• When the user's language is Hebrew or Arabic, NeoOffice would draw text on top of the drop down button in comboboxes and popup buttons
