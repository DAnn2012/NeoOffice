NeoOffice is een compleet pakket kantoorsoftware voor OS X. Met NeoOffice kunnen gebruikers OpenOffice-documenten, PDF-bestanden en ook de meeste Microsoft Word-, Excel- en PowerPoint-documenten bekijken, bewerken en opslaan.

NeoOffice 3.x en OpenOffice gebruikers: voordat u koopt NeoOffice, raden we u klikt op de "NeoOffice Ondersteuning" link en lees de lijst met wijzigingen.

NeoOffice is zeer stabiel en wordt dagelijks gebruikt door duizenden mensen. De ontwikkelaars van NeoOffice hebben NeoOffice gecreëerd in 2003, toen ze OpenOffice voor het eerst draaiend kregen op OS X.

Sindsdien hebben de NeoOffice ingenieurs voortdurend verbeteringen toegevoegd aan NeoOffice die gebruikers niet in vinden in OpenOffice, zoals:

• Bij starten Calc of Impress tonen in plaats van Writer
• OS X Versies
• OS X-eigen grammaticacontrole
• OS X-eigen tekstmarkering
• Systeem-eigen ondersteuning voor blokkeren van bestanden op lokale- en netwerkschijven
• Ondersteuning voor OS X Voorzieningen
• OS X-eigen zwevende hulpvensters


What's New in this Version:
Voordat u koopt NeoOffice, klikt op de "Planamesa Inc. website" link en probeert NeoOffice gratis. This version of NeoOffice adds support for OS X Yosemite's new iCloud Drive feature. We have made signficant changes to NeoOffice's native file locking code to safely coordinate opening, saving, and deleting files in iCloud Drive or files on a shared volume.

This version of NeoOffice also includes fixes for the following bugs:
• NeoOffice would randomly crash when checking if a document has been moved or renamed
• NeoOffice would only display the first image in the many .docx, .xlsx, and .pptx documents
