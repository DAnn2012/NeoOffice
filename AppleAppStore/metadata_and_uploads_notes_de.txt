NeoOffice ist ein vollständiges Office-Paket für OS X. Mit NeoOffice, können Benutzer OpenOffice-Dokumente, PDF Dateien und die meisten Microsoft Word-, Excel- und Powerpoint-Dokumente anzeigen, bearbeiten und speichern.

NeoOffice 3.x und OpenOffice Benutzer: Bevor Sie NeoOffice zu kaufen, empfehlen wir Ihnen, dass Sie auf den „NeoOffice Support” Link klicken und lesen die Liste der Änderungen.

NeoOffice ist extrem stabil und Tausenden von Benutzern nutzen es täglich. Die NeoOffice-ingenieure erstellt NeoOffice im Jahr 2003, wenn sie gemacht OpenOffice ausführen nativ auf OS X zum ersten Mal.

Seitdem sind die NeoOffice-Ingenieure haben immer wieder zu NeoOffice Verbesserungen hinzugefügt, dass die Benutzer nicht werden in OpenOffice zu finden, wie:

• Öffnen von Calc oder Impress anstelle von Writer beim Start mit dem NeoOffice > Beim Starten Öffnen Menü
• OS X Versionen
• Nativen OS X Grammatikprüfung
• Nativen OS X Textmarkierung
• Native Unterstützung für Datei Sperrung auf lokalen und entfernten Volumes
• OS X Dienste Unterstützung
• Native schwebende Werkzeugfenster


What's New in this Version:
Bevor Sie NeoOffice kaufen, klicken auf den „Website von Planamesa Inc.” Link und versuchen NeoOffice kostenlos. This version of NeoOffice adds support for OS X Yosemite's new iCloud Drive feature. We have made significant changes to NeoOffice's native file locking code to safely open, save, or delete files in iCloud Drive, Dropbox, or network drives.

This version of NeoOffice also includes fixes for the following bugs:
• NeoOffice would randomly crash when checking if a document has been moved or renamed
• NeoOffice would only display the first image in many .docx, .xlsx, and .pptx documents
