NeoOffice ist ein vollständiges Office-Paket für OS X. Mit NeoOffice, Benutzer können Text Dokumente, Tabellenkalkulation und Präsentationen lesen, erstellen und bearbeiten. NeoOffice kann Openoffice-Dokumente öffnen und die meisten Microsoft Word-, Excel- und Powerpoint-Dokumente öffnen.

NeoOffice 3.x und Openoffice Benutzer: Wir empfehlen Ihnen klicken Sie auf den NeoOffice Support Link und lesen Sie die Liste der Änderungen, bevor Sie auf diese Version von NeoOffice aktualisieren.

NeoOffice ist extrem stabil und Tausenden von Benutzern nutzen es täglich. Die NeoOffice-ingenieure erstellt NeoOffice im Jahr 2003, wenn sie gemacht OpenOffice ausführen nativ auf OS X zum ersten Mal.

Seitdem sind die NeoOffice-Ingenieure - Patrick Luby und Ed Peterlin - haben immer wieder zu NeoOffice Verbesserungen hinzugefügt, dass die Benutzer nicht werden in Openoffice zu finden, wie:

• Öffnen von Calc oder Impress anstelle von Writer beim Start mit dem NeoOffice > Beim Starten Öffnen Menü
• OS X Gatekeeper
• OS X Versionen
• Native OS X Text Markierung
• Native Unterstützung für Datei Sperrung auf lokalen und entfernten Volumes
• OS X Dienste Unterstützung
• Native schwebende Werkzeugfenster


What's New in this Version:
Diese Version behebt die folgenden Fehler:
• Wiedergabe von Videos in einem Dokument würde manchmal zu NeoOffice zum Absturz
• Exportieren als PDF verursachen würde NeoOffice, den falschen Namen in der Titelleiste des Dokuments angezeigt
