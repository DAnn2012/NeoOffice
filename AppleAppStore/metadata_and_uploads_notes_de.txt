NeoOffice ist ein Office-Paket für Mac, die auf dem OpenOffice Office-Paket basiert. Mit NeoOffice können Sie OpenOffice-Dokumente und LibreOffice-Dokumente anzeigen, bearbeiten und speichern.

Zwei Ingenieure erstellten NeoOffice im Jahr 2003, als sie OpenOffice auf OS X zum ersten Mal native ausführbar machten. Seitdem haben die NeoOffice-Ingenieure immer wieder Verbesserungen zu NeoOffice hinzugefügt, dass Sie nicht in OpenOffice oder LibreOffice finden werden, wie zum Beispiel:

• NeoOffice > Beim Starten Öffnen Menü für die Öffnung von Calc oder Impress anstelle Writer beim Start
• Datei > Alle Versionen durchsuchen Menü für die Wiederherstellung von vorherigen Versionen von Ihrer Dokumente
• Native Datei-Sperrung für die sichere Bearbeitung von Dateien in iCloud Drive, Dropbox oder Netzlaufwerke
• Native Mac Grammatikprüfung
• Native Mac Textmarkierung
• Unterstützung von Mac Dienste
• Native schwebende Werkzeugfenster


What's New in this Version:
Bevor Sie NeoOffice kaufen, klicken Sie bitte auf den „NeoOffice Support” Link und lesen die Installationshinweise.

This version of NeoOffice fixes the following bugs:
• When selecting a menu item while running autospellcheck, NeoOffice would sometimes freeze until autospellcheck finished spellchecking the entire document
• NeoOffice would crash or hang when opening some .rtf files that contain data that does not conform to Microsoft's Rich Text Format specification
• NeoOffice would take 20 - 30 minutes to open spreadsheet documents that contain thousands of unique cell styles
• NeoOffice would sometimes freeze when unlocking a Dropbox or iCloud document in the document's titlebar popup window
• When running NeoOffice on macOS Sierra, both horizontal and vertical scrolling using the trackpad was several times more sensitive than on earlier versions of OS X
• When running NeoOffice on macOS Sierra, NeoOffice would crash when creating a database using the Database Wizard
• NeoOffice would crash if you pressed the Command-w keys while a print dialog was displayed
