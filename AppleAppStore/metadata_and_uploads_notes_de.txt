NeoOffice ist ein vollständiges Office-Paket für OS X. Mit NeoOffice können Benutzer OpenOffice-Dokumente, LibreOffice-Dokumente und einfache Microsoft Word-, Excel- und Powerpoint-Dokumente anzeigen, bearbeiten und speichern.

NeoOffice ist extrem stabil und Tausende Benutzer nutzen es täglich. Die NeoOffice-Ingenieure erstellten NeoOffice im Jahr 2003, als sie OpenOffice auf OS X zum ersten Mal native ausführbar machten.

Seitdem haben die NeoOffice-Ingenieure immer wieder Verbesserungen zu NeoOffice hinzugefügt, die Benutzer nicht in OpenOffice oder LibreOffice finden werden, wie zum Beispiel:

• NeoOffice > Beim Starten Öffnen Menü für die Öffnung von Calc oder Impress anstelle Writer beim Start
• Datei > Alle Versionen durchsuchen Menü für die Wiederherstellung von vorherigen Versionen von Ihrer Dokumente
• Native Datei-Sperrung für die sichere Bearbeitung von Dateien in iCloud Drive, Dropbox oder Netzlaufwerke
• Native OS X Grammatikprüfung
• Native OS X Textmarkierung
• Unterstützung von OS X Dienste
• Native schwebende Werkzeugfenster


What's New in this Version:
Bevor Sie NeoOffice zu kaufen, empfehlen wir Ihnen, dass Sie auf den „NeoOffice Support” Link klicken und lesen die Installationshinweise.

This version of NeoOffice fixes the following bugs:
• When exporting a document to PDF, NeoOffice would not include any characters that use the Apple Color Emoji font
• In compressed and stretched horizontal text, NeoOffice would compress or stretch the text less than the text's "Scale width" setting
• In compressed and stretched vertical text, NeoOffice would compress or stretch Chinese, Japanese, and Korean characters horizontally instead of vertically
• In fontwork objects that have multiple lines of text, NeoOffice would draw all but the first line of text too far to the right
• NeoOffice would sometimes hang when opening .odb files
• When running on OS X Mountain Lion, NeoOffice would ignore changes to the file name in the Save As dialog and would save to the original file name
• When using an 800 x 600 or smaller monitor, NeoOffice would cut off the buttons at the bottom of tall dialogs
