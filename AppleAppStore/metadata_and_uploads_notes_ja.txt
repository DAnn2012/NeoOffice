NeoOfficeのは、OS Xのための完全なオフィススイートです。NeoOfficeのを使用すると、ユーザーは、表示、編集、OpenOfficeの文書、PDFファイル、およびほとんどのMicrosoftのWord、Excel、およびPowerPointのドキュメントを保存することができます。

NeoOffice 3.x and OpenOffice users: before you buy NeoOffice, we recommend that you click on the 「NeoOffice のサポート」 link and read the list of changes.

NeoOffice is extremely stable and thousands of users use it daily. The NeoOffice engineers created NeoOffice in 2003 when they made OpenOffice run natively on OS X for the first time.

Since then, the NeoOffice engineers have continually added improvements to NeoOffice that users will not find in OpenOffice such as:

• NeoOffice > Open at Launch menu to open Calc or Impress instead of Writer at launch
• File > Browse All Versions menu to restore previous versions of your documents
• Native file locking to safely edit files in iCloud Drive, Dropbox, or network drives
• Native OS X grammar checking
• Native OS X text highlighting
• Support for OS X Services
• Native floating tool windows


What's New in this Version:
あなたはNeoOfficeのを購入する前に、「Planamesa Inc. Web サイト」のリンクをクリックして、無料でNeoOfficeのを試してみてください。This version of NeoOffice includes fixes for the following bugs:

• After saving an .odp document, all bullet characters would be hidden when the document was reopened in NeoOffice, OpenOffice, or LibreOffice
• After saving certain .odt documents, some pages would be hidden when the document was reopened in NeoOffice, OpenOffice, or LibreOffice
• After opening an .ods document that contains a link to an unreadable or nonexistent .ods document, clicking on the link would cause NeoOffice to crash
